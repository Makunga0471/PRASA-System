# Template Analysis

| Template Name | Columns | Automation | Agile Suitability |
|--------------|--------|-----------|------------------|
| Basic Kanban | To Do, In Progress, Done | No | Suitable for simple workflows |
| Automated Kanban | To Do, In Progress, Done | Yes (auto-moves issues) | Highly suitable for Agile teams |
| Bug Triage | Open, Investigating, Closed | Yes | Best for managing bugs |

## Justification

I selected the Automated Kanban template because it supports Agile workflows through automation. It automatically moves tasks between stages when their status changes, which reduces manual effort and improves efficiency. This is ideal for the PRASA system where continuous updates, collaboration, and real-time tracking are required.
