# PRASA Smart Digital Monitoring System

## Overview

This project is a REST API system developed using Python and FastAPI for monitoring PRASA operations.

## Features

- Train Management
- Station Management
- Incident Reporting
- REST API
- Swagger Documentation
- CRUD Operations
- Unit Testing

## Technologies Used

- Python
- FastAPI
- Uvicorn
- Pytest
- Pydantic
